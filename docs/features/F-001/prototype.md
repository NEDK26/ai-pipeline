**Status**: Drafted
**Date**: 2026-07-03

## 页面清单

| 编号 | 页面 | 角色 | 目标 | 入口 | 出口 |
|------|------|------|------|------|------|
| P-001 | 演示配置页 | ConfigAdmin | 查看并切换演示模式 | 侧边栏 → System → 演示配置 | 切换成功后留在本页 |
| P-001 | 演示配置页 | Operator | 查看当前演示模式状态 | 侧边栏 → System → 演示配置 | —（只读，无操作出口） |

## 关键用户路径

### 主路径：ConfigAdmin 切换开关

```
侧边栏 System → 演示配置 → 查看当前状态
  → 点击 Switch 切换 → 显示原因输入框 → 填写原因
  → 点击保存 → 确认弹窗 → 点击确认 → 成功提示
  → 页面刷新显示新状态
```

### 权限路径：Operator 只读

```
侧边栏 System → 演示配置 → 查看当前状态
  → Switch 灰色不可点击（disabled）
  → 保存按钮不存在
```

### 异常路径

```
侧边栏 System → 演示配置 → 切换 Switch 后不填原因
  → 点击保存 → 表单校验失败 → 红色提示"修改原因不能为空"
```

```
侧边栏 System → 演示配置 → 切换 Switch → 填原因 → 点击保存
  → 确认弹窗 → 点击取消 → 弹窗关闭 → 开关保持新位置
  （用户可重新点击保存或手动复位）
```

## 页面结构：P-001 演示配置页

### 目标
一眼看到当前状态，一键切换，一因留痕。

### 首屏信息
- 页面标题："演示配置"
- 当前状态：Switch 组件显示 `true`(开启) 或 `false`(关闭)
- 状态辅助文案：`active-text="开启（禁止写操作）"`, `inactive-text="关闭（正常运行）"`

### 核心数据区域
```
┌─ el-card ──────────────────────────────────────┐
│ CardTitle: "演示模式配置"         [保存] 按钮    │
│                                                  │
│  演示模式状态:  [========O] 关闭（正常运行）       │
│                 (el-switch)                      │
│                                                  │
│  修改原因:      [                            ]   │
│                 (el-input type=textarea)         │
│                 0/200 (show-word-limit)           │
│                                                  │
└──────────────────────────────────────────────────┘
```

### 操作与反馈

| 操作 | 元素 | 状态 |
|------|------|------|
| 主操作 | "保存" 按钮 | 值未变: disabled；值已变: enabled |
| 切换 | el-switch | ConfigAdmin: 可点击；Operator: disabled |
| 确认 | ElMessageBox.confirm | 标题"确认操作"，正文含新值和影响说明 |
| 成功 | ElMessage.success | "修改成功" |
| 失败 | ElMessage.error | 显示后端返回的错误信息 |

## 交互与状态

| 状态 | 视觉表现 |
|------|---------|
| 加载中 | 整个 el-card 区域 v-loading |
| 正常显示 | Switch 显示当前值，原因输入框隐藏（无变更时） |
| 已变更未保存 | Switch 新位置，原因输入框显示（动画过渡），保存按钮 enabled |
| 保存中 | 保存按钮 loading 状态，禁止重复点击 |
| 保存成功 | 成功 toast，原因输入框隐藏，保存按钮 disabled |
| 保存失败 | 错误 toast，保持变更状态允许重试 |
| Operator 视图 | Switch disabled + tooltip "仅 ConfigAdmin 角色可修改"，保存按钮不存在 |
| 空状态 | 不适用（始终有值，默认 false） |

## 高风险操作设计

本 Feature 无删除、回滚、批量操作或不可逆操作。开关切换即时生效但可随时切回。

- **二次确认**: 点击保存后弹窗，明确告知切换后果
- **审计记录**: 每次变更写入 `system_operate_log`
- **幂等保护**: 重复设置相同值不产生审计噪音
