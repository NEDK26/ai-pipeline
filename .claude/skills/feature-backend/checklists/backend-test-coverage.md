# 后端测试覆盖检查清单

> 此清单是后端测试覆盖的权威参考。第三步和第五步的测试覆盖要求以此为准。

## 业务规则
- [ ] 主流程正常执行
- [ ] 参数校验（空值、边界、非法值）
- [ ] 权限不足拒绝
- [ ] 状态不允许拒绝
- [ ] 重复提交 / 幂等
- [ ] 并发或条件更新
- [ ] 外部依赖失败

## 状态机
- [ ] 允许转换
- [ ] 禁止转换
- [ ] 失败后状态回滚
- [ ] 回退规则

## 数据
- [ ] 空数据处理
- [ ] 边界数据
- [ ] 字段口径与 api-contract 一致

## 异常
- [ ] 网络或服务失败
- [ ] 依赖不可用
- [ ] 事务回滚

## 高风险操作
- [ ] 发布、回滚、删除二次确认
- [ ] 审计记录生成
- [ ] 权限校验
- [ ] 回滚或恢复

## 测试类型

### 单元测试
- Service 关键规则
- 状态转移
- 权限拒绝
- 参数校验
- 重复调用或幂等
- 审计记录

### 集成 / 接口测试
- Controller 到数据库的主路径
- 请求和返回字段
- 业务错误码
- 权限与状态限制
- 数据库写入结果
- 异常回滚
